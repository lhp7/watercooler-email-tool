# Water Cooler CEO Email Tool

Standalone Streamlit app for preparing Water Cooler at Pegasus Park engagement report email drafts.

## What It Does

- Uploads the report ZIP generated by the keycard reporting app.
- Accepts CEO/contact recipients by Excel or CSV upload, using the Water Cooler CEO list format.
- Fuzzy-matches each recipient organization to the correct PDF report.
- Shows a full editable preview of every email before any draft file is built.
- Shows a downloadable link to the exact PDF attached to each draft.
- Adds Erik Moss's Water Cooler signature and logo to every email draft.
- Generates Outlook `.msg` draft files with the correct PDF attached.
- Produces a downloadable draft log CSV.

```bash
.venv\Scripts\activate

streamlit run app.py
```

```bash
source .venv/bin/activate
```

## CEO Contact Upload Format

The app is compatible with the Water Cooler CEO list Excel/CSV format:

```csv
Primary Contact,Greeting,Contact Title,Contact Email,Organization Name
Jane Doe,Jane,CEO,jane.doe@example.org,Example Organization
```

The `Organization Name` should match the organization name used in the PDF reports. The app uses fuzzy matching, so small spacing or capitalization differences are usually okay.

The app also still supports the older simple CSV format:

```csv
first_name,last_name,email,org_name
Sarah,Johnson,sarah@orgname.org,Texas Trees Foundation
```

## Building Email Drafts

1. Upload the report ZIP from the keycard reporting app.
2. Upload the CEO contact Excel/CSV file or enter recipients manually.
3. Review the matching table and fix any organization names without a matched PDF.
4. Open the attached PDF link for each recipient and edit the full subject/body if needed.
5. Check the final review box.
6. Click `Build reviewed email draft ZIP`.
7. Download the ZIP and unzip it.
8. Select all `.msg` files and drag them into Outlook Drafts.
9. Open Outlook Drafts, review each email, and send one by one.

## Signature

Every generated draft includes:

- Erik's signoff from the body text.
- The Dallas Foundation logo from `signature_logo.png`.
- Erik Moss's title, address, phone numbers, website, and LinkedIn link.

Keep `signature_logo.png` in the same folder as `app.py` and `email_sender.py`.

## Safety Rules

- No draft is generated unless it has a matched PDF report.
- Rows below the match threshold show `No match found` and are blocked.
- Erik must check the final review box before the draft ZIP can be generated.
- This app does not send emails automatically and does not require email account setup.

## Editing the Email Template

Edit `config.py` to change:

- Default reporting period
- Draft sender name/email
- Subject line
- Plain-text body
- Fuzzy match threshold

Available template variables:

- `{first_name}`
- `{org_name}`
- `{period}`
